
# OMDb API app in Angular

 My first app in Angular.

 Task:
 Create simple application with Angular using www.omdbapi.com

## Features

- search field
- list of results
- creating simple persistent wishlist using cookies
- adding to wishlist
- displaying wishlist
- deleting from wishlist
